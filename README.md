# SpunkyBot (Python 3 Port)

This project is a **Python 3 port** of **Spunky Bot**, originally written in Python 2.

Original project:
**Spunky Bot (Python 2)**<br/>
Author: Alexander Kress<br/>
License: MIT License<br/>
Repository: [https://github.com/spunkybot/spunkybot](https://github.com/spunkybot/spunkybot)

This port modernizes the codebase for Python 3 compatibility while preserving the original functionality.

# pyquake3 (Python 3 Port)

This project also includes or relies on a Python 3 port of **pyquake3**, originally developed for interacting with Quake 3 engine–based game servers.

Original repositories:

* gak – [https://github.com/gak/pyquake3](https://github.com/gak/pyquake3)
* urthub – [https://github.com/urthub/pyquake3](https://github.com/urthub/pyquake3)

The Python 3 port adapts the library for modern Python environments while maintaining compatibility with Quake 3–based servers.

# GeoIP Integration

This project uses the `python3-geoip2` library together with **MaxMind GeoIP2** databases to provide IP geolocation features.

GeoIP services and databases are provided by **MaxMind**.

Official resources:

* Website: [https://www.maxmind.com](https://www.maxmind.com)
* GitHub: [https://github.com/maxmind](https://github.com/maxmind)

Available databases include:

* **GeoLite2** (free, registration required)
* **GeoIP2** (commercial version with enhanced accuracy)

Use of MaxMind databases is subject to their licensing terms. Please review the official license agreement before deploying this application.
